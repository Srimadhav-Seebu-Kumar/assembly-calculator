üöÄ Quick Start

Prerequisites

  • Linux x86-64 system
  • NASM (Netwide Assembler)
  • GNU ld (linker)

Installation

# Clone the repository git clone https://github.com/Srimadhav-Seebu-Kumar/assembly-calculator.git cd assembly-calculator # Install NASM (Ubuntu/Debian) sudo apt-get install nasm # Or on CentOS/RHEL/Fedora sudo yum install nasm # or sudo dnf install nasm

Building and Running

# Method 1: Step by step nasm -f elf64 calculator.asm -o calculator.o ld calculator.o -o calculator ./calculator # Method 2: One-line command nasm -f elf64 calculator.asm -o calculator.o && ld calculator.o -o calculator && ./calculator

Simple Assembly Calculator Operations: +, -, *, / Enter first number: 15 Enter operation (+, -, *, /): * Enter second number: 4 Result: 60 Continue? (y/n): y Enter first number: 20 Enter operation (+, -, *, /): / Enter second number: 3 Result: 6 Continue? (y/n): n Goodbye!
